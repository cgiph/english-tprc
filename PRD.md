# Product Requirements Document (PRD)

## 1. Product Overview
The Tradie LMS is a specialized Learning Management System designed to assist skilled migrants (such as mechanics, welders, and carpenters) in preparing for Australian technical assessments and English proficiency exams (PTE). The platform provides targeted technical training, workplace culture bridging, and exam preparation in a single, gamified interface.

## 2. Target Audience
- Skilled tradespeople immigrating to Australia.
- Non-native English speakers needing to pass the PTE.
- Workers needing familiarization with Australian workplace standards, terminology, and safety protocols.

## 3. Key Features
### 3.1. Dashboard & Vision Board
- Personalized dashboard tracking overall progress.
- "Vision Board" showing the user's ultimate goal (e.g., Working in Australia) with visual progress indicators based on their specific technical track and core English requirements.

### 3.2. Course Catalog
- **Technical Courses**: Trade-specific modules (Mechanics, Welding, Carpentry).
- **English Proficiency**: PTE Crash Course and AI Diagnostic tests.
- **Technical English Bridge**: Modules bridging the gap between technical knowledge and Australian workplace communication (including slang and safety reporting).

### 3.3. Interactive Learning & Assessments
- Multimedia lessons (text, mocked videos).
- Multiple-choice quizzes at the end of modules.
- Mocked AI speech analysis for pronunciation and hesitation feedback.

### 3.4. Gamification
- Badge system (e.g., "PTE Ready", "Safety First", "Aussie Slang Master").
- Visual rewards for completing modules and maintaining streaks.

## 4. Technical Requirements (Mockup Phase)
- React-based frontend using Vite and Tailwind CSS.
- Client-side routing using `wouter`.
- Mock data structure for courses, users, and progress state.
- Shadcn UI components for accessible, responsive design.
